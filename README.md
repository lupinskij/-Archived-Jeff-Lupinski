## Installation
Install [node.js & npm](http://blog.npmjs.org/post/85484771375/how-to-install-npm)
```bash
$ npm i -g gulp
```

## Initial Setup
```bash
$ npm i
$ touch .env
```

## Development
```bash
$ npm start
```

## Clean public directory
```bash
$ npm run clean
```

## Build public directory
```bash
$ npm run build
```

## Deployment
Source branch is the development branch because gh-pages needs master to publish
```bash
$ npm run deploy
```
